📘 פרויקט SQL – מערכת נתונים לניהול בוגרים

בפרויקט זה בניתי בסיס נתונים המדמה מערכת ניהול מידע על בוגרי מוסד לימודים.
המערכת כוללת מידע אישי על הבוגר, פרטי העבודה שלו, הכתובת שלו ומספר הילדים, תוך יצירת קשרים לוגיים מלאים בין כל הישויות.

המודל מבוסס על 4 טבלאות מרכזיות המקושרות ביניהן באמצעות מפתחות ראשיים וזרים (PK/FK), ומכיל נתוני דמה מלאים למטרות תרגול וניתוח.

הפרויקט מדגים יכולות של:

בניית סכמת בסיס נתונים מאפס

יצירת קשרים בין טבלאות

שימוש במפתחות ראשיים וזרים

טעינת נתונים (INSERT)

תכנון נכון של מבנה נתונים (Data Modeling)

המערכת משמשת בסיס להמשך עבודה עם שאילתות, ניתוחי נתונים ותרגול של SQL ברמה תפעולית.
